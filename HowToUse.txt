How to use libcommon in your xcode project?
 
1. Run makedist.sh script and pass path to you project. The script created libcommon folder with source code, headers and xcode project. 
Ex: ./makedist.sh ~/Projects/MyNewProject
2. In your project add "libcommon/common.xcodeproj" to project root. You should see "common.xcodeproj" item in your project tree. Expand it and drag libcommon.a to "Target -> <Your Target> -> Link Binary With Libraries".
3. Go to "Project->Edit Project settings" menu item. Add "libcommon/Headers" to "Header search paths"
4. Build your project
